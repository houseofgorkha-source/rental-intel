# RentalIntel Blueprint

**Version:** 0.1 **Status:** Active

## Vision

To become the world's most trusted rental intelligence platform.

## Mission

Help tenants make informed rental decisions through verified reviews and
trustworthy property intelligence.

## Core Principles

-   Trust over growth
-   One RentalIntel ID per real property
-   Users submit properties; RentalIntel verifies them
-   Evidence-backed reviews
-   Data quality over quantity
-   Build for tenants first

## Property Submission Workflow

Property Not Found → Submit Property → Address (Required) → Google Maps
Pin (Required) → Rental Agreement (Optional) → Evidence (Optional) →
Pending Verification → Admin Approval → RentalIntel ID Assigned
